Promotion Icons for VP
Version 1

Promotion Icons for VP (v 1) is the updated version of VP Promotion Icons 1.9 (VPPI) it replaces. Make sure to delete all versions of VPPI in your MOD folder ( \Sid Meier's Civilization 5\MODS )

. 230+ new icons replace triangles and duplicates. New additional symbols and a few colors make promotion icons coherent and also readable
. This mod try to be an improvement for managing and keeping track the units. Nicely
. Enhanced 16x16 format for compatibility with Promotion Flags EUI (v 1.1)
. + 64 Extra Promotion Icons you can use in your own modmods: Extra Promo Icons.png
  IconAtlas = 'extraPromo_Atlas'

------------------------------------------------------------------
Stability

Promotion Icons for VP doesn't add any code mechanics or memory charge when gaming.
Promotions icons included in VP only are affected.

------------------------------------------------------------------
Versions

(v 1)
- Update of VP Promo Icons 1.9 ( VPPI 1.9 ) - "Promotion Icons for VP" replaces "VP Promo Icon"
- Integrated into the Vox Populi installer as a default mod (Thx Gazebo)
- New icons and global rework
- Promotion icons for VPEE and Wonders Expanded are not available anymore (VP promotions only)
- Evaluation: extraPromo_.dds
